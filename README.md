## Start application with: node index.js

[localhost:3456](http://localhost:3456/)
There is a navigation to navigate between pages on every page.

## Routes:

- http://localhost:3456/ : Home
- http://localhost:3456/warframes : Warframes route
- http://localhost:3456/about : About route
- http://localhost:3456/pets : Pets route
- http://localhost:3456/mods : Mods route
  /mods
  /mods?rarity=epic
  /mods?effect=Toxic
  /mods?rank=1
  /mods?rarity=neutral&rank=1
